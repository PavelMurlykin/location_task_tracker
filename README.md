# Задачи рядом

Android-приложение на Kotlin для задач с напоминаниями по местоположению.

## Возможности MVP

- локальный список активных и выполненных задач;
- название, описание и срок задачи;
- привязка координат, адреса и радиуса геозоны;
- выбор точки на Google-карте или ручной ввод координат;
- уведомление при входе в геозону;
- выполнение задачи непосредственно из уведомления;
- восстановление геозон после перезагрузки устройства и обновления приложения;
- настройка интервала между повторными уведомлениями.

## Запуск

1. Установите Android Studio с Android SDK 36 и JDK 17.
2. Создайте файл `local.properties` в корне проекта:

   ```properties
   sdk.dir=C\:\\Users\\USER\\AppData\\Local\\Android\\Sdk
   MAPS_API_KEY=ваш_ключ_Google_Maps
   ```

   `MAPS_API_KEY` необязателен: без него координаты можно указать вручную или
   получить с устройства. Для карты включите Maps SDK for Android в Google Cloud.

3. Откройте проект в Android Studio и запустите конфигурацию `app`.

Или соберите из терминала:

```shell
./gradlew assembleDebug
./gradlew test
```

## Важные разрешения

Геонапоминания требуют точного местоположения и разрешения использовать
местоположение в фоне. На Android 13 и новее уведомления также имеют отдельное
разрешение. Обычные задачи работают и без этих разрешений.

Перед публикацией необходимо заполнить раздел Data safety, добавить политику
конфиденциальности и пройти декларацию фонового доступа к местоположению в
Google Play Console.
